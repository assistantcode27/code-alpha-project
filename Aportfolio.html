tic-tac-toe game

This project is a classic Tic-Tac-Toe game developed using HTML, CSS, and JavaScript. It allows two players to play the game in turns, aiming to align three of their symbols (either X or O) in a row, column, or diagonal. HTML builds the structure, displaying the 3x3 game grid and other elements. CSS styles the board and adds interactive effects, making the game visually appealing. JavaScript handles the game logic, tracking players moves, checking for a winner or a draw, and resetting the game board for new rounds.
